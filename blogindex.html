Question-1: What are the different ways to select an element in the DOM ?

Different ways to select an element in the DOM are getElementsByTagName,getElementById,getElementByClassName,getElementByQuerySelector and querySelectorAll.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where we attach a single event listener to a parent element instead of multiple listeners to individual child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a DOM event propagation mechanism where an event originates from the innermost target element.This process continues until the event reaches the outermost DOM element.
